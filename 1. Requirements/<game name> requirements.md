# 2048 game
3 difficulty modes for the player to choose:
Hard – 4x4 grid (classic 2048)  
Medium – 5x5 grid   
Easy – 6x6 grid
  
  Player can slide tiles Up, Down, Left, or Right.
  After each move, a new tile (2 or 4) spawns randomly in an empty cell.
  When two tiles of the same number collide, they merge into one tile with their sum.
  The current score is displayed and increases by the value of merged tiles.
  Game ends when the player creates a tile with the value 2048 (win) or when no moves are possible (lose).
  Score system: The player starts at zero and gains points by merging tiles; final score is the total accumulated points.
  Highscores saved between sessions for each difficulty mode.
